# Album P2P — Python\n\nSistema P2P de troca de figurinhas digitais para a disciplina de Sistemas Distribuídos.\n\n## Estrutura\n\n```\nalbum-p2p/\n├── index.py                   ← entry point: python index.py\n├── requirements.txt\n├── README.md\n├── config/\n│   ├── peers.json             ← configuração: peer_id, vizinhos\n│   └── inventory.json         ← inventário (lido/escrito em runtime)\n└── src/\n    ├── state.py               ← estado compartilhado em memória\n    ├── inventory.py           ← leitura, escrita e validação do inventário\n    ├── server.py              ← WebSocket server (porta 8080)\n    ├── client.py              ← conexões de saída para vizinhos\n    ├── message_handler.py     ← roteador de tipos de mensagem\n    ├── cli.py                 ← interface de linha de comando\n    └── handlers/\n        ├── hello.py\n        ├── search.py\n        ├── search_hit.py\n        ├── search_miss.py\n        ├── trade_offer.py\n        ├── trade_accept.py\n        ├── trade_reject.py\n        └── transfer_confirm.py\n```\n\n## Configuração\n\nEdite `config/peers.json` com seus dados:\n\n```json\n{\n  "self": {\n    "peer_id": "ALUNO-02",\n    "sticker_id": "FIG-02",\n    "sticker_url": "http://SEU-IP:8080/images/FIG-02.png"\n  },\n  "neighbors": [\n    { "host": "192.168.0.10", "port": 8080 },\n    { "host": "192.168.0.11", "port": 8080 }\n  ]\n}\n```\n\n## Instalação e execução\n\n```bash\npip install -r requirements.txt\npython index.py\n```\n\n## Comandos CLI\n\n| Comando | Descrição |\n|---|---|\n| `inv` | Exibe inventário atual |\n| `search FIG-07` | Busca figurinha na rede por inundação (TTL=7) |\n| `trade ALUNO-05 FIG-02 FIG-07` | Propõe troca: oferece FIG-02, quer FIG-07 |\n| `peers` | Lista peers conectados |\n| `history` | Histórico de trocas realizadas |\n| `help` | Lista todos os comandos |\n| `exit` | Encerra o nó |\n\n## Protocolo\n\n- **Transporte:** WebSocket\n- **Formato:** JSON UTF-8\n- **Porta:** 8080\n- **TTL padrão:** 7\n- **peer_id:** ALUNO-YY\n- **sticker_id:** FIG-XX\n
